# Fallout 4 Tools

A web application featuring interactive tools that replicate mechanics from the game Fallout 4.

## Tools

### Hacking
A word-guessing puzzle solver inspired by Fallout 4's terminal hacking mini-game. Input a list of words and their likeness values (number of characters in the correct position) to determine the actual password.

### Armor Workbench
Browse and explore armor customization options available in Fallout 4's armor workbench. View available modifications and their effects.

### Weapon Workbench
Browse and explore weapon customization options available in Fallout 4's weapon workbench. View available modifications and their effects.
